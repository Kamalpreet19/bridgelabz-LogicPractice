# Invoice Generator Project

This is a Java Maven project for generating cab invoices using TDD.

## Project Structure

src/
├── main/java/com.test
│   ├── App.java
│   ├── InvoiceGenerator.java
│   ├── InvoiceSummary.java
│   └── Ride.java
└── test/java/com.test
├── AppTest.java
└── InvoiceServiceTest.java

## Features
- Calculate total fare
- Calculate average fare
- Compare invoice summaries using equals()
- TDD based development

## How to Run
1. Open in IntelliJ
2. Right click on App.java → Run
3. To run tests: Right click on test folder → Run all tests

## Tech Stack
- Java
- Maven
- JUnit
